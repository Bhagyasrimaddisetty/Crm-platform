# Enterprise CRM Platform

A full-stack Enterprise Customer Relationship Management (CRM) platform inspired by modern enterprise software such as Salesforce. This project is being developed using Java Spring Boot, React, PostgreSQL, Docker, and AWS while following enterprise software engineering practices.

> **Current Status:** Phase 1–3 completed (Project Setup, Database Design, Backend Configuration)

---

## Objectives

* Build a production-style CRM platform
* Learn enterprise backend development using Spring Boot
* Implement secure authentication with Spring Security and JWT
* Develop REST APIs following clean architecture
* Practice testing, CI/CD, monitoring, and cloud deployment
* Gain hands-on experience with enterprise software engineering
